Небольшое пособие для играющих в игру вроде Total Domination, Правила Войны и т.п.
Сортировка точек по расстоянию к данной точке (по возрастанию).
Весь интерфейс руссифицирован (за исключением команд -- их необходимо вводить на English).

Функционал:
- добавление точки* в список;
- удаление точки из списка;
- очистка списка;
- загрузка списка точек из файла;
- вывод списка точек, отсортированного по возрастанию расстояния от них до заданной точки.

* Под словами "добавление точки" подразумевается добавление координат (х;у). С удалением та же история.
